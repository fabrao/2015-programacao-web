HTMl é uma linguagem de marcação utilizada para produzir páginas na Web. Documentos HTML podem ser interpretados por navegadores. As primeiras versões do HTML foram definidas com regras sintáticas flexíveis, o que ajudou aqueles sem familiaridades com a publicação na Web.
Através do tempo, a utilização de ferramentas para autoria de HTML aumentou, assim como a tendencia em tornar a sintexe cada vez mais rigida. Apesar disso (por questões históricas), os navegadores ainda hoje conseguem interprestar paginas web que estão longe de ser um código HTML válido.

Tim Berners-Lee (fisico britânico) criou o HTML original (e outros protocolos associados como o HTTP).
